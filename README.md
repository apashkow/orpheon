# 🎛️ Orpheon

**Natural language → Synth patch generator.**  
Orpheon interprets plain English sound descriptions like:

> _"shimmering ambient pad with a metallic tail"_

...and converts them into structured synth parameters, ready to export as `.vital` patch files (with other formats planned).

---

## 🚀 Features

- 🧠 Prompt interpreter for sound descriptors
- 📖 Descriptor dictionary (`sound_dictionary.json`)
- 🛠 Patch generator stub with export
- 💾 Outputs `.vital.json` patch files (expandable to full `.vital`)
- ✅ Clean Python 3.10+ project structure
